Instructions to run:
1) Open AcumenSystemTests.sln in Visual Studio and build.
2) Install latest Firefox.
3) Install NUnit (https://launchpad.net/nunitv2/trunk/2.6.3/+download/NUnit-2.6.3.msi)
4) Open NUnit runner from the start menu and open project AcumenSystemTests.dll (you built it earlier).
5) Run!

You can also run tests from Visual Studio using a number of common plugins / extesions, including Resharper and "NUnit test adapter".
